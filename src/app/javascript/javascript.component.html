Fonte: Guida ufficiale MDN

JavaScript è case sensitive e utilizza i caratteri Unicode. Le variabili possono essere definite in tre modi:
  • var - variabile base
  • let - varibile locale, block scoped
  • const - costante, block scoped, deve essere inizializzata, non può avere lo stesso nome di una variabile o di una funzione; un oggetto costante può essere modificato nei suoi campi (idem per un array)
L'errore lanciato può essere di qualsiasi tipo, ma conviene usare quelli definti da ES6 o dal DOM.
